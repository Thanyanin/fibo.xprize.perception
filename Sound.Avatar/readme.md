# Sound Avatar

requirement
    -installing python 3.6
    -installing pyaudio


udpsent
    -sending sound  via UDP


udprecieve
    -recieving sound  via UDP


SoundStreaming

CommunicationModule_Server

    -check and change 'Server_IP' with your IPv4 Adress every time (cannot use localhost)
